You can add a span with an ID to a markdown file using the following shortcode: **{{&lt; text_target id=&quot;sample&quot; &gt;}}Text to target{{&lt;/ text_target &gt;}}**

{{< text_target id="sample" >}}Text to target{{</ text_target >}}
